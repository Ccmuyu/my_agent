# Desktop Agent 优化路线图

## 概述

本文档记录 Desktop Agent 的优化方向和扩展计划，按优先级和完成度标记。

---

## 一、已完成

- [x] 多 LLM Provider 支持（OpenRouter）
- [x] 基础工具集（文件、Shell、浏览器、截图）
- [x] RAG 知识库服务
- [x] CLI 和 Web API 模式
- [x] 任务状态机（Pending → Running → Confirm → Complete）
- [x] 执行重试机制
- [x] **流式输出**：任务执行进度实时推送（SSE）

---

## 二、进行中

- [ ] **MCP 协议支持**：实现 MCP SDK，标准化工具扩展
- [ ] **命令白名单**：shell_run 添加危险命令拦截
